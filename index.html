1
display: flex;
display: -webkit-flex;
Назначения: запуск механизма Flexbox
Возможные значения: flex / inline-flex
flex
inline-flex
2
flex-direction: row;
-webkit-flex-direction;row
Назначения: направления укладки элементов по осям
Возможные значения: row / row-reverse / column / column-reverse
row
column
row-reverse
column-reverse
3
flex-wrap: nowrap;
-webkit-flex-wrap;nowrap
Назначения: расположения элементов в одну или несколько строк
Возможные значения: nowrap / wrap / wrap-reverse
nowrap
wrap
wrap-reverse
4
flex-flow: row nowrap;
-webkit-flex-flow: row nowrap;
Назначения: сокращённая запись для свойств flex-direction и flex-wrap
Возможные значения: см. свойства 2 и 3
5
justify-content: flex-start;
-webkit-justify-content: flex-start;
Назначения: выравнивание flex-элементов по главной оси текущей строки
Возможные значения: flex-start / flex-end / space-between / space-around
6
align-items: stretch;
-webkit-align-items: stretch;
Назначения: выравнивание flex-элементов вдоль поперечной оси
Возможные значения: stretch / flex-start / flex-end / center / baseline
7
align-content: stretch;
-webkit-align-content: stretch;
Назначения: выравнивание строк flex-элементов
Возможные значения: stretch / flex-start / flex-end / center / space-between / space-around
